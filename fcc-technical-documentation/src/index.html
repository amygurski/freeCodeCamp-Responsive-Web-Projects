Introduction

C# is an object-oriented programming language that was developed by Microsoft and has been around since 2001.

Tools used:

  • Visual studio: Commonly used for developing C# apps. Community edition is free.
  • Xamarin: Used to develop native Windows iOS and Android apps.
  • Moon Develop: cross platform IDE that allows you to write desktop or web applications
  • Textpad: Text editor for windows. Use libraries, macros, etc.
  • Notepad: Use command-line compiler to compile

    Data Types

    Data types fall into 2 caegories: value types and reference types.

    Value types contain the actual value of the data they store. Common types include int, string, var, DateTime, etc.

    Statements

    Statements are commands meaning they perform some action. For example, they might call a method, calculate something, or declare a value.

    Statements are formed from tokens: keywords, identifiers (variables), operators, and the statement terminator. Note all statements in C# must be terminated with a semi-colon (;).

    Example statement:

    int myInt = 0;

    Identifiers

    <>Identifiers are the sames you give the elements of your program. These can be Namespaces, classes, methods, and variables. Looking at variables, when you create a variable, you want to give it a data type. In this example, we are giving it the integer type: int myInt = 0;

    Identifiers are case-sensitive.

    Operators

    Operators apply operations. For example, they can perform some arithmetic, such as addition or subtraction within the statement.

    mySum = 3 + 4;

    Decision statements

    Loops iterate a task over and over again. For example, this for loop will repeat while i is less than 10:

    for (int i = 0; i < 10; i++) {
